# Ansistrano

## Commandes de déploiement

- `ansistrano deploy` : Déploie l'application en utilisant Ansistrano.
- `ansistrano rollback` : Effectue un rollback de la version déployée précédemment.

## Configuration

- `ansistrano init` : Initialise la configuration Ansistrano dans le projet.
- `ansistrano check` : Vérifie la configuration Ansistrano avant le déploiement.
